# AI Software Company

## Overview
AI Software Company is an AI-powered multi-agent platform that simulates a real software company. It transforms a client's software idea into a complete Software Development Life Cycle (SDLC) by automating requirement analysis, software engineering documentation, project planning, task management, and AI-assisted software generation.

## Features
- AI Requirement Analysis
- Software Requirements Specification (SRS)
- UML Diagram Generation
- User Stories
- Sprint Planning
- Cost Estimation
- Project Dashboard
- Task & Bug Tracking
- AI-Assisted Software Generation
- Progress Monitoring

## Tech Stack
- Frontend: React.js
- Backend: Express.js
- Database: MongoDB
- AI: OpenAI / Claude / Gemini APIs
